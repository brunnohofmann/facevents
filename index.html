Facevents

Insira a id do evento que deseja interagir e a opção desejada.

Sobre

Facevents é uma biblioteca que utiliza a API do Facebook para exibir os perfis relacionados a um evento específico em seu site.
Essa ferramenta basicamente trata dados em JSON e exibe-os em seu site.

Para que os participantes sejam exibidos, o evento deve ser público.

Instalação

Credenciais do Facebook

Para poder utilizar o Facevents você precisa criar uma conta de desenvolvedor do Facebook

PAra isso visite o site https://developers.facebook.com/ e crie um novo app.

Na aplicação você deve informar o APP Id a App Secret, esses dois dados formarão o token de segurança exigido pelo Facebook para exibir os dados de evento.

JavaScript

1 - Baixa os arquivos em Javascript

2 - Insira o arquivo JS no seu HTML

<script src="facevents.js"></script>

facevents(
//para saber os dados a seguir consulte a documentação
//sua id facebook
"SUA_APP_ID",
//sua chave de acesso facebook
"SUA APP_SECRET"
);
